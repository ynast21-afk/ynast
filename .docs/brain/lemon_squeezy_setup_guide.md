# 🍋 Lemon Squeezy 사장님용 가이드 (왕초보 버전)

사장님, 이 가이드를 보시고 **딱 3가지 정보**만 저에게 주시면 됩니다. 최대한 상세히 썼으니 그대로 따라만 오세요!

---

## 🚀 1단계: 회원가입 및 내 가게 이름 정하기
1. **[Lemon Squeezy 회원가입](https://app.lemonsqueezy.com/register)** 페이지로 들어갑니다.
2. 이메일과 비밀번호를 넣고 가입한 후, 메일함에 가서 **인증 버튼**을 누릅니다.
3. 처음 로그인하면 **Store Setup** 화면이 뜹니다.
   - **Store Name**: `StreamVault` 라고 적으시면 됩니다.
   - **Country**: `South Korea` 선택
   - **Currency**: `United States Dollar (USD)` 선택 (중요! 달러여야 정산이 편합니다.)

## 📦 2단계: 판매할 상품(자유이용권) 등록하기
1. 왼쪽 메뉴에서 **Products** 아이콘을 누릅니다.
2. 오른쪽 상단의 **(+) New Product** 버튼을 누릅니다.
3. 아래 항목을 채워주세요.
   - **Name**: `VIP 멤버십` 또는 `VIP Access`
   - **Description**: `프리미엄 영상을 무제한으로 시청하는 멤버십입니다.`
   - **Pricing Model**: **Subscription** (이걸 골라야 매달 자동 결제가 됩니다.)
   - **Price**: `$8.00` (원하시는 가격)
   - **Billing Period**: `Monthly` (한 달에 한 번 결제)
4. 맨 위 오른쪽의 **Publish product** 버튼을 눌러서 저장합니다.

## 🗝️ 3단계: 저에게 주셔야 할 3가지 찾기 (핵심!)

### 번호 1. API Key (비밀 통로 열쇠)
1. 왼쪽 맨 아래 **Settings** (톱니바퀴) -> **API** 메뉴로 들어갑니다.
2. **(+) New API Key** 버튼을 누릅니다.
3. 이름은 마음대로(예: `AFK`) 지어주시면 됩니다.
4. 화면에 긴 문자가 나오면 바로 **복사**해서 저에게 주세요. (한 번만 보여주니 꼭 복사해두세요!)

### 번호 2. Store ID (가게 번호)
1. **Settings** -> **Stores** 메뉴로 들어갑니다.
2. 사장님이 만드신 `StreamVault` 이름 옆에 **ID: 12345** 같은 숫자가 보일 겁니다. 그 숫자를 알려주세요.

### 번호 3. Variant ID (상품 진짜 번호)
1. 다시 **Products** 메뉴로 돌아가서 아까 만든 상품을 클릭합니다.
2. 화면을 조금 아래로 내리면 **Variants**라는 칸이 보입니다.
3. 그 옆에 있는 **ID: 67890** 같은 숫자를 저에게 알려주세요.
